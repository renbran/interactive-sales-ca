# AI Role-Play Training System

## Overview
The AI Role-Play Training System allows sales agents to practice their skills in a safe, realistic environment with AI-powered prospects. The system uses GPT-4 to simulate realistic prospect personas with varying levels of difficulty, objections, and communication styles.

## Features

### üé≠ 8 Realistic Prospect Personas

1. **Eager Student** (Easy)
   - Excited and ready to go
   - Asks logistics questions
   - Quick to make decisions
   - Best for: New agents learning basics

2. **Skeptical Parent** (Hard)
   - Concerned about costs, safety, quality
   - Asks probing questions
   - Wants proof and guarantees
   - Best for: Advanced objection handling practice

3. **Budget-Conscious Prospect** (Medium)
   - Focused on affordability
   - Repeatedly asks about costs
   - Interested in scholarships
   - Best for: Price objection handling

4. **Indecisive Prospect** (Medium)
   - Asks many questions
   - Circles back to same topics
   - Needs reassurance and guidance
   - Best for: Consultative selling practice

5. **Experienced Researcher** (Expert)
   - Very detailed questions
   - Wants specifics and data
   - Highly technical
   - Best for: Advanced agents, complex scenarios

6. **Competitive Shopper** (Hard)
   - Mentions competitors
   - Asks "what makes you different?"
   - Focused on getting best deal
   - Best for: Competitive differentiation practice

7. **Career-Focused** (Hard)
   - Wants ROI calculations
   - Asks about placements and outcomes
   - Business-like communication
   - Best for: Value proposition practice

8. **Visa-Worried** (Medium)
   - Anxious about visa rejection
   - Asks many visa-related questions
   - Had previous rejection
   - Best for: Handling anxious prospects

## How It Works

### 1. Setup
- Enter your OpenAI API key (stored locally, only sent to OpenAI)
- Select a prospect persona to practice with
- Click to start the practice session

### 2. During the Session
- **Real-time Conversation**: Type or use voice input to respond
- **AI Responses**: GPT-4 generates realistic prospect responses based on:
  - Persona personality traits
  - Conversation history
  - Current phase (opening, discovery, presentation, objection-handling, closing)
  - Objections raised and handled

- **Live Coaching Panel**: Shows:
  - Current conversation phase
  - Real-time coaching hints
  - Objection tracking (raised vs. handled)
  - Suggestions and warnings

- **Voice Support**:
  - Speech recognition for voice input (Chrome, Edge)
  - Text-to-speech for AI responses
  - Natural conversation flow

### 3. After the Session
- **Performance Metrics**:
  - Overall Score (0-100%)
  - Script Adherence
  - Objection Handling
  - Rapport Building
  - Closing Effectiveness

- **Detailed Feedback**:
  - Strengths identified
  - Areas to improve
  - Key moments in conversation
  - Objections encountered and quality of handling
  - Recommended training modules

## Performance Scoring

### Script Adherence (0-100%)
- Measures how closely you followed the sales script
- Tracks references to script sections
- Rewards structured approach

### Objection Handling (0-100%)
- Tracks objections raised by prospect
- Measures how many were addressed
- Evaluates quality of responses

### Rapport (0-100%)
- Analyzes sentiment of prospect responses
- Tracks positive vs. negative reactions
- Measures relationship building

### Closing (0-100%)
- Evaluates if conversation reached closing phase
- Assesses outcome quality
- Rewards positive close

## Best Practices

### For New Agents
1. Start with "Eager Student" persona (Easy difficulty)
2. Focus on script adherence first
3. Practice the complete flow: Opening ‚Üí Discovery ‚Üí Presentation ‚Üí Close
4. Use coaching hints actively
5. Review performance metrics after each session

### For Experienced Agents
1. Challenge yourself with "Skeptical Parent" or "Competitive Shopper" (Hard)
2. Practice complex objection scenarios
3. Work on rapport building with difficult personas
4. Try "Experienced Researcher" (Expert) for technical practice
5. Aim for 80%+ overall scores

### Voice Practice Tips
1. Speak clearly and naturally
2. Pause after prospect speaks (wait for AI response)
3. Use the microphone button for voice input
4. Practice as if it's a real phone call
5. Pay attention to tone and pacing
